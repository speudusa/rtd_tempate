Exercise Solutions: Controllers and Routes

Line numbers are for reference. They may not match your code exactly.

Part 1: GET Request

Modify your HelloController class to display a form on a GET request that asks the user for both their name and the language they would like to be greeted in.

 1   //GET: /<controller>/
 2   public IActionResult Index()
 3   {
 4      string html = "<form method='post'>" +
 5         "<input type='text' name='name' />" +
 6         "<select name='language'>" +
 7         "<option value='english' selected>English</option>" +
 8         "<option value='french'>French</option></select>" +
 9         // ... add any other languages here ...
10         "<input type='submit' value='Greet Me!'/>" +
11         "</form>";
12
13      return Content(html, "text/html");
14   }

Part 2: POST Request

  1. Include a new public static method, CreateMessage, in the HelloController that takes a name as well as a language string. Based on the language string, you’ll display the proper greeting.

 1   public static string CreateMessage(string name, string language)
 2   {
 3      string helloTranslation = "";
 4      switch (language)
 5      {
 6         case "french":
 7            helloTranslation = "Bonjour ";
 8            break;
 9         case "english":
10            helloTranslation = "Hello ";
11            break;
12         // ... add any other languages here ...
13      }
14      return helloTranslation + name;
15   }
