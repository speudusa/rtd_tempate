Line numbers are for reference. They may not match your code exactly.

Class Design

This is one example of how 1 base class (Computer) and 2 derived classes (Laptop & Smartphone) can use inheritance.

  1. Using the diagram above, set up the base class: Computer.

  • Testing the Computer class.

 1   public class Computer
 2   {
 3      public double Ram { get; set; }
 4      public double Storage { get; set; }
 5      public readonly bool hasKeyboard;
 6
 7      public Computer(double ram, double storage, bool hasKeyboard)
 8      {
 9         Ram = ram;
10         Storage = storage;
11         this.hasKeyboard = hasKeyboard;
12      }
13
14      public double IncreaseRam(double extraRam)
15      {
16         return Ram += extraRam;
17      }
18
19      public double IncreaseStorage(double extraStorage)
20      {
21      return Storage += extraStorage;
22      }
23   }

  • Use inheritance to create the Laptop derived class.

 1   public class Laptop : Computer
 2   {
 3      public double Weight { get; set; }
 4
 5      public Laptop(double ram, double storage, bool hasKeyboard, double weight) : base(ram, storage, hasKeyboard)
 6      {
 7      Weight = weight;
 8      }
 9
10      public bool IsClunky()
11      {
12         if (Weight > 5.0)
13         {
14            return true;
15         }
16            else
17         {
18            return false;
19         }
20      }
21   }

Class Implementation

  1. Add a new MSTest project to your solution.

1   // one Computer class tests in the Computer Class
2   [TestMethod]
3   public void TestIncreasingRam()
4   {
5      Computer testingComputer = new Computer(2, 3, true);
6      Assert.AreEqual(2, testingComputer.Ram);
7      testingComputer.IncreaseRam(3);
8      Assert.AreEqual(5, testingComputer.Ram);
9   }

  1. Try to add three MSTest tests to each class. Consider testing each method or field.

  • Testing the Smartphone class

1   //Smartphone Class
2   [TestMethod]
3   public void TestTakingSelfies()
4   {
5      SmartPhone testingSmartphone = new SmartPhone(2, 3, true, 800);
6      testingSmartphone.TakeSelfie();
7      Assert.AreEqual(801, testingSmartphone.NumberOfSelfies);
8   }

Abstract class design

  1. Create the AbstractEntity Class.

 1   // AbstractEntity Class
 2   public class AbstractEntity
 3   {
 4      public int Id { get; set; }
 5      private static int nextId = 1;
 6
 7      public AbstractEntity()
 8      {
 9         Id = nextId;
10         nextId++;
11      }
12   }

  1. Update the Computer class. Remember Computer extends AbstractEntity.

public class Computer : AbstractEntity

Testing AbstractEntity using MSTest:

  1. Testing the Computer Class

 1   //Computer Class
 2   [TestMethod]
 3   public void TestInheritsId()
 4   {
