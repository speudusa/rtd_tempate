AddGrade Method

This method accepts two parameters—a number of course credits and a numerical grade (0.0-4.0).

1   public void AddGrade(int courseCredits, double grade)
2   {
3      // Update the appropriate properties: NumberOfCredits, Gpa
4      double totalQualityScore = Gpa * NumberOfCredits;
5      totalQualityScore += courseCredits * grade;
6      NumberOfCredits += courseCredits;
7      Gpa = totalQualityScore / NumberOfCredits;
8   }
